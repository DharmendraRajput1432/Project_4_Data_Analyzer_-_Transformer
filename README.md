# Data Analyzer and Transformer

A menu-driven Python project that demonstrates the use of Python fundamentals such as functions, recursion, lambda expressions, global variables, sorting, filtering, built-in functions, `*args`, `**kwargs`, and 1D/2D list manipulation.

## Features

### 1. Input Data

* Accept user input for a 1D list.
* Accept user input for a 2D list (matrix).

### 2. Sample Dataset

* Load predefined sample data for quick testing.

### 3. Display Data

* Display all stored 1D and 2D data.
* Print 2D data in tabular format.

### 4. Recursion

* Calculate Factorial using recursion.
* Generate Fibonacci series using recursion.

### 5. Data Filtering

* Filter 1D list values based on a threshold.
* Filter 2D list values based on a threshold using lambda functions.

### 6. Data Sorting

* Sort 1D list in ascending or descending order.
* Sort each row of a 2D list in ascending or descending order.

### 7. Statistical Analysis

For 1D data:

* Total Elements
* Minimum Value
* Maximum Value
* Sum
* Average

For 2D data:

* Rows and Columns
* Minimum Value
* Maximum Value
* Sum of Elements

### 8. Unique Values

* Display unique values from a 1D list.
* Display unique values from each row of a 2D list.

### 9. Average Calculation

* Calculate average of 1D list elements.
* Calculate average of each row in a 2D list.

### 10. *args and **kwargs Demonstration

* Use `*args` and `**kwargs` concepts for displaying dataset statistics and summaries.
